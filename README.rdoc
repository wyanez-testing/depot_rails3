== Depot Rails 3 Application

Depot es una aplicación usada como tutorial en el libro "Agile Web Development with Rails, 4th Edition, Rails 3.2". Permite manejar un catalogo de productos y un carrito de compras. Para mayor descripción de sus funcionalidades se recomienda revisar el capitulo 5 del libro.
